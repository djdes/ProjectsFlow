# Поведение Dashboard

## Навигация

Dashboard сохраняет левый AI-чат на desktop/tablet. На мобильном sidebar заменяется одним dropdown разделов, а AI-чат скрывается.

## Разделы

### Overview

Название, иконка, описание, дата создания, usage, visibility, приглашение пользователей и platform badge.

### Users

Users/Pending requests, поиск, фильтр роли, приглашение и таблица имя/роль/email.

### Data

- карточки сущностей и поиск таблиц;
- сортировка A–Z;
- schema-aware grid;
- поиск по полям;
- горизонтальная и вертикальная прокрутка;
- typed filters;
- permissions CRUD;
- Add item;
- клик строки открывает правую редактируемую панель;
- пустое состояние таблицы.

Наблюдались сущности Category, Order и Product. Product содержал 12 строк, Order — одну, Category — пустая.

### Analytics

Live visitors и аналитика; в неопубликованном приложении отображается пустое состояние.

### Marketing

SEO & GEO с Overview/Meta tags/Advanced Settings и генератор social content. Это не приоритет ProjectsFlow.

### Domains

Встроенный URL, копирование/редактирование, custom domain и email domain/sender.

### Integrations

Skills, connectors, OAuth connectors; часть функций закрыта тарифом.

### Security

Security scan и advanced settings.

### Code

Дерево файлов и split view; редактирование кода закрыто тарифом.

### Agents / Workflows

Предлагаемые агенты и автоматизации по расписанию/триггеру. Не нужны для первого этапа ProjectsFlow.

### Logs

Refresh, email filter, categories, events, errors-only, type/user/timestamp и раскрываемая строка деталей.

### API

Генерируемая документация, JS SDK, authentication, CRUD endpoints, OpenAPI и copy for LLM.

### Settings

App metadata, main page, visibility, badge, clone/template, test data, session recordings, delete; Authentication для email/password, Google, Microsoft, Facebook, Apple и SSO.
